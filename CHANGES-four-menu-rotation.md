# 4メニュー順番ローテーション変更メモ

## 概要

- 既定の進行を8日周期から「肩・腕 → 脚 → 胸 → 背中」の順番ローテーションへ変更。
- 曜日固定ではなく、実施したメニューを基準に次メニューへ進む。
- 休みの日は「休み」を選び、必要な日は今日画面から別メニューへ変更できる。

## メイン種目

- 肩・腕: ショルダープレス
- 脚: スクワット
- 胸: ベンチプレス
- 背中: ハーフデッドと床引きデッドを背中メニューの実施回数で交互に切替

メイン種目は原則3セット5回。完遂で次回+2.5kg、未達なら同重量、同じ重量で2回連続未達なら約10%減を提案する。

## 互換性

- 旧8日ローテのログ、推定1RM履歴、MAX測定結果は削除しない。
- 新規4メニューログには `fourMenuRotation` と実施メニュー情報を追加して保存する。
- JSONエクスポート/インポートは旧バックアップでも不足キーを既定値で補完する。

## PWA

- `index.html` のJS/CSSバージョンと Service Worker のキャッシュ名を更新。
